# Java-homework

Swing 电子宠物小项目，练习面向对象、定时任务、GUI 绘制与简单持久化。

## 功能概览
- 多宠物系统：内置小龙/小猫/小狗三种性格，支持自定义名字，领养的每只宠物都会保存进度。
- 养成状态：饱食度、心情、体力在界面显示并会随时间衰减；照顾得好会进化，忽略太久会生病。
- 交互操作：喂食、玩耍、睡觉三个按钮，对状态有不同影响，同时记录操作日志与次数。
- 动画与提示：宠物会在窗口中来回飞行，根据心情/状态切换颜色、表情，并显示提示语。
- 存档：退出后会将全部宠物状态写入 `~/.flying_pet_state.properties`，下次启动自动恢复。

## 运行方式
```bash
javac $(find src -name '*.java')
java -cp src petgame.App
```

如果使用 IDE，可直接在 `petgame.App` 的 `main` 方法上运行。首次打开会自动创建一只小龙，可通过“新增宠物”按钮扩展更多伙伴。记得定期喂养、陪玩和睡觉，解锁闪耀进化形态！
